# Senko_Pica
Projekts par picērijas funkcionalitāti. Tiek izmantoti dialoga logi JOptionPane, pogas ar JComboBox un teksta lauks ar JTextField. Iespējamās darbības, piemēram, reģistrēt klientu, veikt pasūtījumu utt.
